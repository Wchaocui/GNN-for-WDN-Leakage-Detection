# GNN-for-WDN-Leakage-Detection
This repository contains the code for the paper Hierarchical Self-Attention Graph Pooling Networks for Leakage Detection of Water Distribution Networks.
Full paper link: https://ieeexplore.ieee.org/document/10606924
The experimental dataset is sourced from the public Water Distribution System Research Database: https://www.uky.edu/WDST/database.html
We select the KY1 network as the water distribution network (WDN) test case.
